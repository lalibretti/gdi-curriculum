Course Goals

Gain an understanding of what code editors are used for and what features are inside.

Why use an editor?

We need a place to write our code in a way that's readable for human eyes. A code editor will help us do that.

We want to write and store code in a way that is easy for the human eye to interpret

What code editor do I choose?

Picking a code editor is like choosing between a android or apple- once you pick one it's hard to switch.

Personal anecdote trying to switch code editors for this class. It's really hard to switch!! but you can see them if you open the speaker notes window (hit 's' on your keyboard).

Why would I choose VS Code?

Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded Git.

Visual Studio Code vs Visual Studio

Visual Studio is NOT the same. Visual Studio is an Integrated Development Environment (IDE)--meaning you can write and test your code at the same time. It's a great tool but it runs slow.

What's Emmet?

A set of plug-ins for text editors that allow for high-speed coding and editing in HTML, XML, XSLT, and other structured code.
In Visual Studio Code Emmet is pre-installed. In Atom you have to download the package.
